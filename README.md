**Overview**

This project is a simple ASP.NET Core MVC web application built to demonstrate the core structure of an MVC-based web application. It showcases routing, controller logic, view rendering, configuration management, and static asset handling.

The goal of this project is to demonstrate a clean and organized web application structure using .NET.

**Technologies Used**

ASP.NET Core

C#

.NET 8

MVC Architecture

Razor Views

CSS

**Project Structure**

Program.cs → Application entry point and middleware configuration

WebsiteApp.csproj → Project configuration file

HomeController.cs → Handles routing and request processing

Views/ → Razor view pages

_ViewStart.cshtml → Shared layout configuration

wwwroot/site.css → Static styling files

**Key Concepts Demonstrated**

MVC Architecture (Model-View-Controller)

Routing and Controller Actions

Razor View Engine

Middleware configuration in Program.cs

Static file serving

Application configuration via appsettings.json

**How to Run the Project**

Clone the repository

Open the solution file (.sln) in Visual Studio

Restore NuGet packages

Run using IIS Express or Kestrel
